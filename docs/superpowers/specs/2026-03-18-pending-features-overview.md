## 三、已确认的冲突决策

### 1. R1 与 C2：不足单元模式应用层级 ✅
- **确认结果**:
  - 内层（自然时段边界）：应用不足单元判断（FULL/PROPORTIONAL）
  - 外层（相对时间段边界）：直接截断，截断部分按不足单元判断
- **配置位置**: 规则级别配置，应用于两种边界

### 2. R1 与 P3：双层封顶 vs 周期延长 ✅
- **确认结果**: 采用"气泡弹开"模型
  - 免费时段像水管中的气泡，将计费时间"弹开"
  - 气泡前的段：边界不变
  - 气泡后的段：边界整体后移 = 气泡长度

### 3. R1 与 O1：长期简化与双层结构 ✅
- **确认结果**:
  - 简化金额 = 周期封顶金额
  - 前提条件：该周期内无任何优惠

### 4. P3 与 O1：周期延长与长期简化 ✅
- **确认结果**:
  - 周期延长型免费时段本身就是优惠
  - 第一版实现：有优惠就不能简化，必须逐周期计算
